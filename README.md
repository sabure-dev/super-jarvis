# SuperMarket Together: Голосовой инструмент установки цен #


## Установка ##
- Склонить репозиторий:
```git clone https://github.com/sabure-dev/super-jarvis```

- Создать виртуальное окружение:
```uv venv```

- Установить зависимости:
```uv pip install . ```

- Запустить main.py:
```uv run main.py```
